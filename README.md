I'm **Farizy** — a self-taught engineer operating at the intersection of AI development, offensive security research, and open-source software. Every project I've shipped was built from a mobile device, without formal education, and without a team. I believe advanced tools should be free, autonomous, and owned by the people who use them.

Currently focused on building AI-native CLI tooling, on-device intelligence, and security research frameworks — all open, all yours.
